# justmd 需求文档

> 极简桌面 Markdown 编辑器

## 产品定位

一个基于 Tauri + React + TypeScript 的极简 Markdown 编辑器，单文件模式，零打扰的编辑体验。

---

## 编辑模式

- **三态切换**：纯编辑 / 编辑+预览 / 纯预览
- 切换方式：右上角按钮 + 菜单（`Ctrl+1/2/3`）
- **分屏布局**：左右分屏，比例可拖拽调整（默认 50/50）
- **滚动同步**：简化版，按最近标题锚点同步

## 编辑区

- 使用原生 `<textarea>`，无语法高亮，无行号
- **不引入** textarea 增强插件（Tab 缩进、自动配对等）——保持纯原生体验
- **搜索替换**：`Ctrl+F` / `Ctrl+H`，纯字符串操作，支持区分大小写
  - 查找下一个、替换、全部替换
- 字体：系统等宽字体

## 预览区

- 渲染引擎：`react-markdown` + `remark-gfm`
- **代码高亮**：Shiki（VS Code 同款引擎），主题随暗/亮切换
- **数学公式**：KaTeX
- **Mermaid 图表**：支持
- **不支持图片**渲染
- 预览样式：GitHub 风格 CSS

## 文件管理

- **单文件模式**，无多 tab
- 文件名显示在窗口标题栏
- 底部状态栏仅显示字数统计

### 文件操作

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 新建 | `Ctrl+N` | 清空当前，变为未命名文件 |
| 打开 | `Ctrl+O` | 系统对话框选择 `.md` / `.markdown` |
| 保存 | `Ctrl+S` | 已关联文件直接保存，未命名文件走"另存为" |
| 另存为 | `Ctrl+Shift+S` | 系统对话框选择保存位置 |

### 拖拽打开

- 支持从文件管理器拖拽 `.md` / `.markdown` 到窗口
- **在当前窗口打开**，替换当前文件（有自动保存，不丢数据）
- 拖拽时窗口有视觉反馈（虚线边框高亮）
- 拖拽非支持格式时，底部状态栏提示"不支持的文件类型"

### 最近文件

- 文件菜单含"打开最近"子菜单，保留 **8 个**历史记录
- 启动时自动将最近文件同步到原生菜单
- **不自动清理死链**：点击已删除文件时，弹窗提示"文件不存在"，然后从列表移除
- 数据存储在 localStorage

## 自动保存

- 已关联文件：停止输入 **1.5 秒**后自动写入磁盘
- 未命名文件：编辑内容自动缓存到 localStorage
- 首次 `Ctrl+S` 时触发"另存为"对话框
- **不显示保存状态指示器**（消除焦虑）

## 外部修改检测

- 每 3 秒检测一次当前文件是否被外部修改
- 若检测到差异，弹窗提示"文件已在外部修改，是否重新加载？"

## 主题

- **默认跟随系统**
- 支持手动切换：浅色 / 深色 / 自动
- 切换按钮在右上角（`Ctrl+D`）
- 预览区代码高亮、Mermaid 图表主题同步跟随
- 偏好存储在 localStorage

## 启动行为

- **恢复上次会话**：
  - 重新打开上次编辑的文件
  - 未命名文件从 localStorage 恢复缓存内容
  - 恢复编辑模式、分屏比例
- 若上次文件已不存在：降级为空白文档，状态栏提示

## 界面框架

- **自定义标题栏**（隐藏系统标题栏，`decorations: false`）
- 单行工具栏，左侧菜单按钮，右侧控制按钮：
  - 左侧：文件下拉菜单（新建/打开/打开最近/保存/另存为）、视图下拉菜单（模式切换/主题切换）、justmd 标识
  - 中间：当前文件名
  - 右侧：主题切换按钮、模式切换按钮、窗口控制按钮（最小化/最大化/关闭）
- 标题栏整体可作为窗口拖拽区域
- 窗口默认尺寸：1200×800

## 数据持久化

| 数据 | 存储位置 |
|------|---------|
| 会话状态（文件路径、模式、比例、主题） | localStorage |
| 未命名文件缓存内容 | localStorage |
| 最近文件列表（8个） | localStorage |
| 主题偏好 | localStorage |

## 不支持的功能（明确排除）

- 多 tab / 多文件同时编辑
- 图片插入与渲染
- 导出 PDF / HTML
- 正则表达式搜索
- 代码折叠、行号、语法高亮（编辑区）
- 插件系统
- 同步/云服务

## 技术栈

| 层级 | 技术 |
|------|------|
| 桌面框架 | Tauri v2 |
| 前端框架 | React 19 + TypeScript |
| 构建工具 | Vite |
| Markdown 渲染 | react-markdown + remark-gfm |
| 数学公式 | remark-math + rehype-katex |
| Mermaid | mermaid (客户端渲染) |
| 代码高亮 | Shiki |
| 存储 | localStorage |
